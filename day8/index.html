Intro to Web Development

Fundamentals of Web Development

HTML

HTML stands for Hyper Text Markup Language. This is the language used to create websites and web applications. It is used to describe the structure of a webpage. HTML is analogous to the bones of the human body.

CSS

CSS stands for Cascading Style Sheets. CSS is used to describe the visual representation of webpages. It is used for determining things such as layout, colours, and fonts of webpages. CSS is analogous to the skin of the human body.

JavaScript

JavaScript is another fundamental part of web development. It is used to add interactivity to webpages to make websites dynamic. JavaScript is analogous to the brain of the human body.
